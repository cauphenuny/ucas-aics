本章为基于PyTorch的模型实现，该章节共包含基于VGG19实现图像分类、实时风格迁移的推断、实时风格迁移的训练、自定义PyTorch CPU算子、Transformer架构实现与应用四个实验。

（1）基于VGG19实现图像分类对应着exp_4_1_VGG19；

（2）实时风格迁移的推断对应着exp_4_2_StyleTransfer_infer；

（3）实时风格迁移的训练对应着exp_4_3_StyleTransfer_train；

（4）自定义PyTorch CPU算子对应着exp_4_4_mysigmoid；

（5）Transformer架构实现与应用对应着exp_4_5_transformer。
